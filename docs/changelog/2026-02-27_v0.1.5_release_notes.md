# OpenFlux v0.1.5 Release Notes

**Created:** 2026-02-27  
**Author:** Development Team  
**Status:** Published  

## Overview

OpenFlux v0.1.5 adds sandboxed process management (kill/list), a "Learn Skill" preset workflow, redesigned typing indicator with intent text display, multiple Agent and UI bug fixes, and open-source preparation (bilingual README, LICENSE, .gitignore, etc.).

---

## New Features

### 1. Process tool: kill / list commands

**Impact:** Sandboxed process management  
**Description:** Added `kill` and `list` actions to the process tool for full lifecycle management of Agent-spawned background processes.
- New `SpawnedProcess` interface tracking spawned processes (PID, command, start time, session ID)
- `kill`: Terminate a process by PID
- `list`: List all active background processes

**Modified:** `gateway/src/tools/process/index.ts`, `gateway/src/tools/common.ts`

---

### 2. "Learn Skill" preset workflow

**Impact:** Workflow system  
**Description:** Added 9th preset workflow "Learn New Skill" — search, download and install skills from online skill hub (OpenClaw/ClawHub), converting them to locally executable workflows.  
**Triggers:** "learn skill", "install skill", etc.

**Modified:** `gateway/src/workflow/presets.ts`

---

## Bug Fixes

### 3. Fix empty messages causing LLM API 400 error

**Impact:** Multi-agent routing  
**Trigger:** Empty messages in session history cause LLM API 400 errors  
**Fix:** Filter empty messages when loading session history (`content.trim().length > 0`)

**Modified:** `gateway/src/agent/manager.ts`

---

### 4. Fix file-only uploads not showing in session

**Impact:** Chat functionality  
**Trigger:** Uploading files without text input shows no message in session list  
**Fix:** Auto-generate `[Uploaded: filename]` as message content when user has no text but has attachments

**Modified:** `gateway/src/agent/manager.ts`

---

## Improvements

### 5. Typing indicator redesign

**Impact:** Chat UI  
**Description:** Replaced fire-blackhole spinning animation with bouncing dots effect. Added LLM intent/thinking text display (truncated to 120 chars). Typing indicator position correctly restored after session switch.  
**New functions:** `ensureTypingPosition()`, `updateTypingText()`

**Modified:** `src/main.ts`, `src/styles/main.css`

---

### 6. Open-source preparation

- Bilingual README (English + Chinese)
- MIT LICENSE
- Comprehensive .gitignore (excludes sensitive data, runtime files, large binaries)
- Git LFS for ONNX model files
- Removed debug scripts (gen_icons.py, gen_nsis_images.py)
- Added openflux.io website badge

---

- Direct overlay installation, no additional configuration needed
- Existing configs are not affected
